CSS继承即子标记会继承父标记的所有样式风格，并且可以在父标记样式风格的基础上再加以修改，产生新的样式，而子标记的样式风格完全不影响父标记。 值得注意的是，并不是所有的属性都会自动传给子元素，有的属性不会继承父标记的属性值，例如边框属性就是非继承的。
CSS的全称是“层叠样式表”，层叠特性和继承不一样，可以把层叠特性理解成“冲突”的解决方案，即对同一内容设置了多个不同类型样式产生冲突时的处理，CSS规定如下的优先级：
行内样式>id样式>class样式>标记样式

这是body的文本内容。
body {
font-size: 12px;
/* 元素样式 */
}

.c1 {
font-size: 28px;
color: blue;
font-family: "黑体";
/* class样式 */
}

#p1,
#p2 {
font-family: "幼圆";
font-size: 36px;
/* id样式 */
}

第一段 子标记p继承了父标记body的样式。

第二、三、四段都设置了class="c1"。此时p_font-size: 200%无效

第三段设置了id="p1"。此时p_font-size: 200%无效；c1全部无效，因为c1_color: blue无效

行内样式style="font-family:Arial, Helvetica, sans-serif;color:red;"，优先级最高。 此时p_font-size: 200%无效

class="c1";id="";行内样式style="font-family:Arial, Helvetica, sans-serif;color:red;"，优先级最高。 此时p_font-size: 200%无效;c1字体大小有效

class="";id="p2";行内样式style="font-family:Arial, Helvetica, sans-serif;color:red;"，优先级最高。 此时p_font-size: 200%无效;p2字体大小有效

class="";id="p1";行内样式style="font-family:Arial, Helvetica, sans-serif;color:red;"，优先级最高。 此时p_font-size: 200%无效;p1字体大小有效

class="";id="";行内样式style="font-size:12px;font-family:Arial, Helvetica, sans-serif;color:red;"，优先级最高。 此时p_font-size: 200%无效

class="";id="";行内样式style="font-family:Arial, Helvetica, sans-serif;color:red;"，优先级最高。 此时p_font-size: 200%有效

class="c1";id="";行内样式style="font-family:Arial, Helvetica, sans-serif;"，优先级最高。 此时p_font-size: 200%无效

class="c1";id="p1";行内样式style="font-family:Arial, Helvetica, sans-serif;"，优先级最高。 此时p_font-size: 200%无效;p1_size有效;c1_color有效;?第三段?
